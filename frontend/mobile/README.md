# React Native PPT Generator

This is the React Native mobile application for PPT generation.

## Setup

1. Install dependencies:
```bash
npm install
```

2. For iOS:
```bash
cd ios && pod install && cd ..
npx react-native run-ios
```

3. For Android:
```bash
npx react-native run-android
```

## Shared Logic

The UI components and API logic are shared between web and mobile apps.
See `src/shared/` for shared components and services.
